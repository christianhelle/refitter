## 3. Enumerations

### MultipleInterfaces
- `Unset` - Do not generate multiple interfaces
- `ByEndpoint` - Generate a Refit interface for each endpoint with a single Execute() method
- `ByTag` - Generate a Refit interface for each tag

### OperationNameGeneratorTypes
- `Default` - Use the Refitter IOperationNameGenerator implementation (uses MultipleClientsFromOperationId by default but if there are duplicate operation ID's it is changed to MultipleClientsFromFirstTagAndPathSegments)
- `MultipleClientsFromOperationId` - Generates multiple clients and operation names based on the Swagger operation ID (underscore separated)
- `MultipleClientsFromPathSegments` - Generates the client and operation name based on the path segments (operation name = last segment, client name = second to last segment)
- `MultipleClientsFromFirstTagAndOperationId` - Generates the client name based on the first tag and operation name based on the operation id
- `MultipleClientsFromFirstTagAndOperationName` - Generates the client name based on the first tag and operation names based on the Swagger operation ID (underscore separated)
- `MultipleClientsFromFirstTagAndPathSegments` - Generates the client name based on the first tag and operation name based on the path segments
- `SingleClientFromOperationId` - Generates the client and operation name based on the Swagger operation ID
- `SingleClientFromPathSegments` - Generates the operation name from path segments (suffixed by HTTP operation name if need be)

### TypeAccessibility
- `Public` - Type is accessible by any assembly that references it
- `Internal` - Type is only accessible within its own assembly

### IntegerType
- `Int32` - Use System.Int32 (int) for integers without format
- `Int64` - Use System.Int64 (long) for integers without format

### CollectionFormat
- `Multi` - Multiple parameter instances rather than multiple values (?param=value1&param=value2)
- `Csv` - Comma-separated values (param=value1,value2,value3)
- `Ssv` - Space-separated values (param=value1 value2 value3)
- `Tsv` - Tab-separated values (param=value1\tvalue2\tvalue3)
- `Pipes` - Pipe-separated values (param=value1|value2|value3)

### AuthenticationHeaderStyle
- `None` - Do not generate any "Authorization" header attributes
- `Method` - Generate a "Authorization" header as a method attribute
- `Parameter` - Generate a "Authorization" header as a method parameter attribute

### CacheProviderType (Apizr)
- `None` - No cache provider
- `Akavache` - Akavache cache provider
- `MonkeyCache` - MonkeyCache cache provider
- `InMemory` - In-memory cache provider
- `DistributedAsString` - Distributed cache as string provider
- `DistributedAsByteArray` - Distributed cache as byte array provider

### MappingProviderType (Apizr)
- `None` - No mapping provider
- `AutoMapper` - AutoMapper mapping provider
- `Mapster` - Mapster mapping provider

### TransientErrorHandler (Dependency Injection)
- `None` - No transient error handling
- `Polly` - Polly Framework and HTTP Extensions
- `HttpResilience` - Microsoft HTTP Resilience Library

## 4. Features NOT in CLI but Available in .refitter File

These features are ONLY available via `.refitter` file configuration:

1. **Multiple OpenAPI Documents Merge** (`openApiPaths`) - Merge multiple OpenAPI specs into a single client
2. **Naming Settings** (`naming.useOpenApiTitle`, `naming.interfaceName`) - Control interface naming
3. **Status Code Comments** (`generateStatusCodeComments`) - Control whether status codes are documented in XML comments
4. **Content-Type Headers** (`addContentTypeHeaders`) - Control generation of Content-Type headers (no --no-content-type-headers CLI flag exists)
5. **Response Type Override** (`responseTypeOverride`) - Dictionary to override return types for specific operations
6. **Optional Parameters** (`optionalParameters`) - Re-order optional parameters (different from `--optional-nullable-parameters`)
7. **Dependency Injection Settings** (entire `dependencyInjectionSettings` object) - No CLI flags for any DI configuration
8. **Apizr Settings** (entire `apizrSettings` object except `withRequestOptions`) - No CLI flags for cache provider, mapping provider, priority, mediation, file transfer
9. **Code Generator Settings** (entire `codeGeneratorSettings` object except `integerType` and `inlineJsonConverters`) - Most NSwag type customizations unavailable via CLI
10. **Contract Type Suffix** (`contractTypeSuffix`) - Append suffix to all contract type names

- Schema trimming/tree-shaking
