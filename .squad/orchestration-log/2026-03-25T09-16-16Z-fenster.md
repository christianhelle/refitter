## Key Decisions
1. **Simple enum over polymorphic JSON:** `PropertyNamingPolicy` is a serializable enum (not a factory/interface), enabling instant parity across CLI, `.refitter`, Source Generator, and MSBuild without polymorphic JSON complexity.
2. **Preserve original strategy:** Invalid C# identifiers are minimally sanitized (underscores for invalid chars, prefix for invalid starts). Reserved keywords are escaped with `@`. Valid identifiers pass through unchanged.
3. **`[JsonPropertyName]` always emitted:** Ensures serialization correctness regardless of property naming policy.
